to the Project (whether via GitLab, GitHub mirror, email, or any other
channel), You agree that the Contribution is licensed under the terms of
this CLA.

For Contributions to be accepted, your commit messages must include the
following sign-off line (matching the email address used by your Git
identity):

```

```

This line can be added automatically with `git commit -s`. The sign-off
constitutes Your acceptance of this CLA for the Contribution it accompanies.

If You contribute on behalf of an employer or other entity, a separate
Corporate CLA may be required. Contact the Project Owner via
<https://gitlab.com/MisterJB/x265-butler> to arrange one.
